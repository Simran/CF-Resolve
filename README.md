CF-Resolve
=====================

CF-Resolve is a simple app to attempt to retrieve the actual IP of a website that is protected by CloudFlare services. 

*Coded using MonoDevelop on Ubuntu 12.04

*Works on Windows, Linux, and MacOSX.
